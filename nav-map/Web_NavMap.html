Notes

  • Every guarded page redirects to /Login when Session Role is missing or wrong.
  • Admin links in _Layout.cshtml visible only when Session IsAdmin == 1, set at login.
  • Login decides role via srv.CheckUserAdmin Username, which returns true if user exists in Teacher table.
  • Web does not have register or SignUp UI. New users register via WPF or MAUI clients.
