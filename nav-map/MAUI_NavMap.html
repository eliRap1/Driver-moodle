Notes

  • Routing uses Shell.Current.GoToAsync. Routes registered in AppShell.xaml.
  • Shell.TabBarIsVisible=False hides the auto tab bar so role pages do not bleed into each other.
  • Each page calls AppState.RequireRoleAsync in OnAppearing. Wrong role redirects to its own home or to Login.
  • Login routes by CheckUserAdminAsync = true → Teacher, else Student. Admin flag separate via IsUserAdminAsync, stored in AppState.IsAdmin for the ADMIN badge on TeacherHome.
  • MAUI does not yet have Admin pages or AllStudents or AllTeachers. Future addition.
